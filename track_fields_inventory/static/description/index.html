Create History After Editing Inventory

This module simply help to write history of changes after User editing product inside inventory.

It will be saved inside chatter or Odoo logs upon saving.

Installation Procedure

  1. Download this module and put it inside your odoo addons file.
  2. From Odoo interface, enter the debug mode > Apps menu > update apps.
  3. In the search bar, delete everything and type in Edit History of Inventory
  4. Restart your Odoo service
  5. Refresh your browser, now the module will work in product template after editing and saving
    Note : if there's error when you are about to install, try to restart your odoo services and then begin your installation.
